    MCMC module documentation¶

    1. 数据同化方法¶

    数据同化方法最初来源于数值天气预报，为数值天气预报提供初始场的数据处理技术。数据同化方法基于贝叶斯原理，基本思路是将不同来源、不同分辨率观测数据融合入模型，反向求得最优参数，以提高模型的模拟精度和预测能力，参数的后验概率分布可以从参数的先验知识以及模拟和观测结果的似然度求得： 其中 \theta 为参数，Z 为观测值，p(\theta|Z) 是在已经观测值时参数的后验概率分布，p(\theta) 表示参数的先验概率密度函数（Probability density functions, PDF），在此模块中设定为最大值和最小值之间的均匀分布。p(Z) 为观测数据的概率密度函数，p(Z|\theta) 为观测数据在先验参数值下的条件概率密度分布函数，也称为参数 \theta 的似然函数，所有待估计参数的初始取值范围根据已有研究确定。

    本模块中假设每个参数均独立于所有其他参数，并具有均值为0的对数正态分布：

    p(Z|\theta) \propto exp\begin{Bmatrix}{-\sum \frac{[Z_i-\phi_i \times X]^2}{2\sigma_i^2}} \end{Bmatrix}

    其中 Z_i 表示观测数据集中第 i 个土壤观测值的对数， X 是从模型中得到的碳库的对数，可以通过假设当前土壤状态处于稳态而得出， \phi 是将模拟碳库映射到观测值的映射向量。我们将土壤碳库的误差设定为：

    \sigma = 0.5 \times Z_i

    参数后验分布空间的计算采取Metropolis-Hastings （简称M-H）算法。M-H算法的核心是通过马氏链蒙特卡洛（简称MCMC）方法随机采样得到模型参数的高维后验分布。M-H算法涉及到重复的两个步骤，一个是产生参数的步骤，一个是判断参数是否被接受的步骤。基于上一步接受的参数以及一个可能的概率分布来产生一个新的参数。\theta_{max} 和 \theta_{min} 分别代表参数的最大值和最小值：

    \theta_{new} = \theta_{old} + r \times (\theta_{max} - \theta_{min})/D

    其中 \theta_{old} 和 \theta_{new} 分别是之前接受的参数和新产生的参数。r 是在-0.5 到 0.5 之间的随机变量，D用来控制移动步长在本模块中我们根据Xu et al.（2006）的结果将其设为5。在每一个产生参数的步骤中，新产生的参数根据Metropolis 指标判断是否接受这一参数，这一指标是根据新参数的似然度函数相比上一个接受的参数的似然度函数是否有所提高。

    条件反演是分阶段利用贝叶斯概率反演过程的方法，原理是将上一阶段已约束的参数固定（即取其最优值）作为已知参数，然后进入下一阶段反演过程，依次顺序，直至没有更多待优化参数被约束为止。

    2. 数据来源¶

    本模块中我们所用来评价和限制模型的全球土壤有机碳数据来自于Harmonized World Soil Database (HWSD) 数据库。HWSD的土壤特性代表了在成岩发育、土地利用和在干扰历史下各个阶段的真实土壤剖面数据。我们意识到土壤是一个动态且非均质的环境，因此HWSD的观测结果在当前的气候、植被和土地管理的活动下不一定处于平衡态。然而本模块中我们的目标是优化模型参数，评估模型是否模拟了与现实世界中观测结果大致相似的土壤碳库。HWSD代表了最详尽、最全面的土壤特征数据库，是可用于评估生态系统模型中土壤生物地球化学模块的有力资源。

    HWSD的土壤包括表层土壤有机碳（0-30cm）和深层土壤有机碳（30-100cm）。考虑到HWSD原始数据的分辨率和模型分辨率不一致，因此本模块中我们使用了由Wieder et al. (2014) 等人由HWSD原始分辨率（30 arc sec）插值到CLM模型分辨率的数据产品用来进行模型的限制和评估。（数据来源：https://daac.ornl.gov/SOILS/guides/HWSD.html)

                        soilcarbon_ss_emulater[0:3, ilat, ilon] = np.matmul(inv_AKs, -I[:, ilat, ilon])

        soilcarbon_ss_emulater_total = np.sum(soilcarbon_ss_emulater, axis=0)
         soilcarbon_ss_emulater_total = np.where(soilcarbon_ss_emulater_total == 0, np.nan, soilcarbon_ss_emulater_total)

        log_soilcarbon_M_total = np.log10(soilcarbon_ss_emulater_total)
        scalar_sd = 0.5
        diff = log_soilcarbon_M_total - log_soilC_tot_area_glob
        J = (diff ** 2) / (2 * (scalar_sd * log_soilC_tot_area_glob) ** 2)
        J_new = np.nansum(np.nansum(J))
        delta_J = J_new - J_old

        if min(1, np.exp(-delta_J)) > random.random():
            Parameters_keep[upgrade, :] = par_new
            J_keep[upgrade] = J_new
            upgrade += 1
            par_old = par_new
            J_old = J_new
            soilcarbon_ss_emulater_total1 = np.where(soilcarbon_ss_emulater_total > 0, soilcarbon_ss_emulater_total,np.nan)
            count_recode[upgrade] = np.nansum(np.nansum(soilcarbon_ss_emulater_total1, axis=1), axis=0)
            ss_recode[upgrade] = soilcarbon_ss_emulater_total_area_global

        Parameters_rec[simu, :] = par_old

    4. 小结¶

    土壤有机碳库是陆地生态系统最大的碳库，储存在土壤中的碳是植被和大气中所储存碳总量的三倍。因此准确模拟陆地生态系统土壤碳库是反映陆地生态系统和大气碳净气体交换的关键。然而现阶段不同模型对土壤碳库的模拟存在5-6倍的差异，模拟的结果存在巨大的不确定性。模型参数作为模型中的基础部分，是其不确定性的重要来源。本模块中应用数据同化方法，基于观测值对模型参数进行限制，使得模型参数在很大程度上被优化，进而更为准确的模拟土壤碳库。
